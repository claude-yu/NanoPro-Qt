SciEdit / NanoPro —— 科研图编辑器（便携版）
================================================

【怎么打开】
1. 把整个 NanoPro 文件夹解压出来（不要只复制 NanoPro.exe，它需要旁边的文件）。
2. 双击 NanoPro.exe 即可运行，无需安装 Python 或任何环境。
3. 第一次打开若弹出 Windows 蓝色拦截（“Windows 已保护你的电脑”）：
   点【更多信息】→【仍要运行】。这是因为程序没做数字签名，不是病毒。

【它能做什么】
- 导入图片/PDF，分图层编辑（显隐、锁定、打组、拖拽排序、不透明度）。
- 选区工具：矩形/套索/魔棒/选区画笔、去背景、自动拆解、GrabCut 抠图。
- 画笔/橡皮/文字/矢量钢笔/锚点编辑；标尺、参考线、对齐分布。
- 导出 PNG / TIFF / SVG / PDF（投稿可达 300 DPI）；保存/加载工程(.nanopro.json)。
- AI 生图 + AI 对话生成提示词（需自己的 API Key，见下）。

【AI 功能要不要配】
- 不配也能用上面所有的常规编辑功能。
- 「AI 抠图（去背景）」内置了本地模型，开箱即用、离线、无需联网或 Key：
  插件菜单 → AI 抠图设置 → 后端选「本地内置模型」即可。难抠的彩色复杂图可改用云端 grsai/Qwen（需 Key）。
- 想用「生图 / 对话」：在面板「设置」里选商家、填自己的 API Key、保存即可。
  · 生图走 grsai 或自定义中转站；对话走 DeepSeek / 智谱 GLM / 自定义中转。
  · 没 Key 时面板里有「前往注册获取」的链接。
  · Key 只存在你自己电脑的 C:\Users\你的用户名\.sciedit\config.json，不上传、不进程序。

【可选：PDF 矢量导入更准】
- 想把 PDF 当矢量导入编辑，电脑里最好有 poppler(pdftocairo) 或 Inkscape 任一；
  没有也能用，只是退化为按页转图。

【设置 / 缓存放哪】
- 所有设置、布局、Key 都在：C:\Users\你的用户名\.sciedit\
- 想彻底重置：关掉程序后删掉这个文件夹即可。

【遇到问题】
- 打不开 / 闪退：确认是 64 位 Windows 10/11；把整个文件夹放在英文路径下（避免特殊字符）。
- 反馈优化意见时，告诉我：哪一步、点了什么、期望什么、实际什么。
